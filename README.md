# Estimasi tingkat penggunaan media sosial pada mahasiswa Program Studi Statistika Universitas Mataram menggunakan metode Two Stage Cluster Sampling.
# Latar belakang
Perkembangan teknologi informasi yang semakin pesat telah menjadikan media sosial sebagai bagian penting dalam kehidupan sehari-hari, khususnya di kalangan mahasiswa. Media sosial tidak hanya digunakan sebagai sarana komunikasi, tetapi juga sebagai media hiburan, akses informasi, hingga mendukung aktivitas akademik. Tingginya intensitas penggunaan media sosial di kalangan mahasiswa dapat memberikan dampak positif, seperti kemudahan memperoleh informasi pembelajaran, namun juga dapat menimbulkan dampak negatif apabila penggunaannya berlebihan dan mengganggu konsentrasi belajar.  

Mahasiswa Program Studi Statistika Universitas Mataram merupakan salah satu kelompok yang aktif memanfaatkan media sosial dalam aktivitas sehari-hari. Tingkat penggunaan media sosial pada mahasiswa menjadi hal yang menarik untuk dikaji karena dapat memberikan gambaran mengenai pola penggunaan media sosial dalam lingkungan akademik. Untuk memperoleh gambaran tersebut diperlukan suatu proses pengambilan sampel yang tepat agar estimasi yang dihasilkan dapat merepresentasikan populasi secara baik.  

Dalam penelitian ini digunakan metode **Two Stage Cluster Sampling**, yaitu teknik sampling dua tahap yang dilakukan dengan memilih kelompok atau *cluster* pada tahap pertama, kemudian memilih sampel individu pada *cluster* terpilih pada tahap kedua. Pemilihan metode ini dilakukan karena populasi mahasiswa secara alami telah terbagi ke dalam kelompok berdasarkan angkatan dan kelas, sehingga metode *cluster sampling* dinilai lebih efisien dibandingkan metode sampling acak sederhana. Berdasarkan hal tersebut, penelitian ini dilakukan untuk mengestimasi tingkat penggunaan media sosial pada mahasiswa Program Studi Statistika Universitas Mataram dengan menggunakan metode **Two Stage Cluster Sampling**. Data penelitian dikumpulkan melalui penyebaran kuesioner menggunakan Google Form dengan skala Likert, kemudian dianalisis untuk memperoleh estimasi karakteristik populasi berdasarkan desain sampling yang telah ditentukan.

# Tujuan Penelitian
Mengukur tingkat penggunaan media sosial pada mahasiswa Program Studi Statistika Universitas Mataram berdasarkan data hasil penyebaran kuesioner yang telah disusun menggunakan skala Likert.
Menguji kualitas instrumen penelitian melalui uji validitas dan reliabilitas terhadap butir pertanyaan yang digunakan agar instrumen layak dipakai dalam pengambilan data penelitian.
Menerapkan teknik Two Stage Cluster Sampling dalam proses penarikan sampel, yaitu dengan melakukan pemilihan cluster pada tahap pertama berdasarkan kelompok mahasiswa, kemudian memilih responden pada cluster terpilih pada tahap kedua.
Menghitung probabilitas pemilihan dan bobot sampling berdasarkan desain pengambilan sampel yang digunakan agar estimasi yang dihasilkan dapat merepresentasikan populasi dengan baik.
Melakukan estimasi parameter populasi berupa rata-rata tingkat penggunaan media sosial mahasiswa dengan menghitung nilai estimasi, standard error (SE), interval kepercayaan 95%, relative standard error (RSE), serta design effect (Deff).
Membandingkan hasil estimasi antara metode Simple Random Sampling (SRS) dan Two Stage Cluster Sampling untuk melihat efisiensi desain sampling yang digunakan berdasarkan nilai estimasi, standard error, dan design effect.
Menyusun kesimpulan dan rekomendasi berbasis hasil analisis terkait gambaran tingkat penggunaan media sosial mahasiswa serta memberikan interpretasi hasil yang sesuai dengan data penelitian.
